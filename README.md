> A React component implementing the latest version of Animate.css

## Install

```bash
npm install @charlesvien/react-animatecss
```
or alternatively
```bash
yarn add @charlesvien/react-animatecss
```

## Usage

```ts
import Animate from '@charlesvien/react-animatecss';

<Animate>
  Magnolia
</Animate>
```
